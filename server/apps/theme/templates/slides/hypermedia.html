Hypermedia ( Hypertext )

A nonlinear medium of information that includes graphics, audio, video, plain text and hyperlinks.

  • 1963 Hypertext Editing System - Brown University
  • 1987 HyperCard - Apple
  • 1990 HTML 1.0 Links
  • 1995 HTML 2.0 Forms
  • 1996 IFRAME - Internet Explorer
  • 1999 XMLHTTP - Microsoft
  • 2006 XHR W3C Standard
